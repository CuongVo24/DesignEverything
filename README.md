# DesignEverything

> Bộ **Lõi-text portable** (kịch bản phỏng vấn + template doc + taxonomy + lý do) **+ adapter native gầy theo từng harness** — biến câu trả lời đời thường của người mới thành một thư mục tài liệu nền móng chuẩn. Ra mắt từ Claude Code.

**Không build agent.** Chạy trên lưng agent có sẵn (Claude Code, Codex, Cursor, Antigravity...). Đối tượng: người hoàn toàn không biết viết doc, dự án từ số 0. Cốt lõi là **phỏng vấn**, không sinh từ template trống.

## Nguyên tắc sống còn
**Lõi phải béo, adapter phải gầy.** Portable ⟷ Enforce kéo nhau: hứa nội dung như nhau mọi nơi, ép tốt nhất nền tảng cho phép (graceful degradation).

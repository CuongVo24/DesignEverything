## Tại sao cần file này
File này giúp chốt dự án là gì và vì sao nó đáng tồn tại, trước khi mọi người lao vào tính năng hay công nghệ. Nếu thiếu file này, scope rất dễ phình vì ai cũng hiểu sản phẩm theo một kiểu khác nhau.

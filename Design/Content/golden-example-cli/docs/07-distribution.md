## Tại sao cần file này
File này định nghĩa phương án đóng gói, phân phối và cách thức người dùng cuối (chủ yếu là các lập trình viên hoặc DevOps) cài đặt và cập nhật công cụ dòng lệnh này. Bỏ qua bước này sẽ khiến công cụ khó tiếp cận và khó tích hợp vào các hệ thống tự động hóa.

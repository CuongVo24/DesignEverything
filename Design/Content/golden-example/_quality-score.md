# Golden Example — Quality Score

## Tại sao cần file này
File này chấm lại toàn bộ cây `docs/` mẫu theo Quality Rubric để batch sau có chuẩn vàng mà bám. Nó giúp phân biệt "đủ file" với "đủ chất lượng".

## Ghi chú chấm điểm
- Bộ docs bám đúng một dự án cụ thể, không viết chung chung.
- Scope có MoSCoW rõ và Must là tập nhỏ nhất chạy được.
- Nhánh mobile đã bới ra sớm hai bẫy: offline/sync và quy trình lên store.
- Tất cả file đều có `## Tại sao cần file này` và anchor `status=planned`.

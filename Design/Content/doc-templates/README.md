# Doc Templates — Mẫu file đầu ra

> Mỗi file trong [taxonomy](../taxonomy.md) có một template. Template = khung heading + đoạn **"Tại sao cần file này"** + chỗ chèn câu trả lời đã dịch ngược + mỏ neo truy vết.

## Tại sao cần file này
Đây là phần "đóng gói" giá trị: câu trả lời đời thường của người dùng được rót vào một khung chuẩn, chuyên nghiệp, có giải thích. Template quyết định doc cuối nhìn ra sao.

## Quy ước mọi template
1. Mở đầu bằng `## Tại sao cần file này` (1–2 câu, ngôn ngữ người mới).
2. Nội dung được agent **dịch ngược** từ câu trả lời (không bê nguyên văn đời thường).
3. Cuối mỗi mục có mỏ neo theo [../../Core/AnchorFormat.md](../../Core/AnchorFormat.md).
4. Ngắn — đủ dựng nền móng, không làm người mới ngợp.
